# FastAPI Hotel Booking System 🏨

## 🚀 Project Overview
This is a FastAPI backend project for managing hotel room bookings.

## 🔧 Features
- REST APIs using FastAPI
- Pydantic validation
- CRUD operations (Create, Update, Delete)
- Booking workflow (Booking → Check-in → Check-out)
- Search, Sorting, Pagination
- API testing using Swagger UI
